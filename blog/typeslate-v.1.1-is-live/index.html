TypeSlate v1.1: Markdown Export, Session Recovery, Themes & More

July 10, 2025 · By Josh

TypeSlate v1.1 is Live

The first update of TypeSlate is live. I launched the program a month ago and have continued using it every day. In that time, I made notes — features I wished it had, bugs I noticed, and ways to make it better.

I think this update delivers on that. I’m excited about this version. The newest release of TypeSlate includes a few major updates that I’m especially excited to share.

Save as Markdown

Focus Better with TypeSlate

TypeSlate is a fullscreen writing app that locks out distractions and helps you finish what you start.

Download Free

I wanted to create a way for Obsidian users to augment their workflow, not interrupt it. For writers who already organize their notes in Obsidian, I wanted an easy way to integrate TypeSlate into their writing process. That turned out to be surprisingly simple: just add an option to save as . md .

Now, all you have to do is point your save folder to your Obsidian vault, and when you’re finished writing, you’ll see your draft appear in Obsidian — ready to use.

Themes

I added a few different variations of background and text colors to appeal to more people. Instead of a simple dark mode toggle, you now have a dropdown with all available options.

Import Last Session

This feature came from personal experience. I was interrupted during writing sessions but didn’t want to start over with a blank page. Now you can select Load Previous Session , and the program will import the text from your last saved session into the new one. Word counts still start at zero — just like with the import from clipboard or import from template tools.

Bug Fixes

When I originally packaged the program, I didn’t notice that the spellchecker library and export-to-Excel libraries weren’t loading properly. As a result, spellcheck and the export functions didn’t work in the Microsoft Store version. That’s now fixed.

Icon

Microsoft does some weird stuff with MSIX packaging and icons. Even transparent ones sometimes end up with a colored background. Microsoft gave my taskbar icon a blue background — and I hated it.

I’ve changed it to a full square icon, like the one featured in this post’s image.

Upload Stats Window

The upload stats windows were not configured correctly and opened too small. That’s now fixed — they’re properly sized and readable.

Update v1.1 in the Books

I’m proud of this most recent update. TypeSlate continues to evolve into a unique word capturer — something in a class of its own. It’s not trying to replace your word processor. I built this program as a first-pass drafting tool — something to help you write without interruptions.

That remains the core philosophy. And this version only strengthens that foundation.

The best part about TypeSlate is that it’s free. Not freemium. Not “free while supplies last.” Just free.

Why? Because the world needs your words. And because I had a lot of fun building this — and I want to share it with like-minded people.
